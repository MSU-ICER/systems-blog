Current Maintenance Activities

Development Node Configuration Change

Apr 13, 2026

On Tuesday, April 14th, at 8:00PM, we will be applying a configuration change to the development nodes in order to increase system stability. This change will reserve a portion of memory solely for system processes. This will mitigate situations where high memory demand could otherwise cause development nodes to become unresponsive. This configuration change is expected to be non-disruptive, however, if you encounter any issues or have any questions, please contact us

Network Maintenance 4/9/2026

Apr 8, 2026

On Thursday, April 9, 2026, between 9AM and 11AM we will be installing two new network switches. No impact to running jobs is expected. If you notice any issues with connectivity or running jobs between 9AM and 11AM please open a ticket.

Research File System Quota Troubleshooting Beginning 4/9/2026

Apr 6, 2026

Beginning Thursday, April 9, 2026, we will be working with our vendor to troubleshoot a technical issue with the Research Space quota system. No outage is anticipated; however, you may notice periodic slowdowns while we troubleshoot this issue. This message will be updated as troubleshooting continues, and will be resolved once the quota system is returned to normal operation. If you experience any issues accessing a Research Space or Project directory during this time, please open a ticket.

Operating System Updates Beginning 4/1/2026

Apr 1, 2026

Over the next several weeks, ICER will be performing minor operating system updates across all nodes in the HPCC. During this time, users may notice longer queue times and some nodes unavailable in Slurm while they are being updated. Once the updates are complete, nodes will be returned to service and jobs will continue to schedule and run. Because these are minor updates, the software and module system is unaffected. No issues have been identified in initial testing, but please open a ticket if you experience any issues.

Current Announcements

HPCC Scheduled Maintenance

Apr 9, 2026

The HPCC will be unavailable on Thursday, May 7 starting at 5AM for regularly scheduled maintenance. No jobs will run during this time.

Jobs that will not be completed before this date will not begin until after maintenance is complete. For example, if you submit a four day job three days before the maintenance outage, your job will be postponed and will not begin to run until after maintenance is completed.

If you have any questions, please contact us

Miniforge3 version deprecation

Feb 25, 2026

What is happening?

On Monday, May 4, 2026, the current version of the Miniforge3 module (which provides access to the conda command), 24.3.0-0 will be removed. All users should update to using the new version 25.11.0-1.
