Updated: OnDemand Portal Update on Thursday 6/11

  • Jun 10, 2026 • Categories: announcement maintenance • Status: open

Update 6/12 9:00am: The new OnDemand server is now available. Some users may still get directed to the old server due to caching of the server address. Users will get directed to the new server as their caches expire.

At 8:00PM on Thursday, June 11th, ICER’s OnDemand portal will undergo a major version update to version 4.2. This update introduces several new features, including:

  • An option for interactive session notifications
  • A new System Status application to view overall cluster resource utilization
  • The ability to save, edit, and delete interactive application settings
  • An updated file editor interface
  • The display of both requested cores and GPUs on session cards
  • The ability to download session information as a zip file to attach to a support request

If you have any questions about this update or encounter any issues with the OnDemand portal following the update, please contact us.
